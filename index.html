Slot Machine Game - Let's Play!

A simple JavaScript based HTML5 Slot Machine Game:

User Interface/Functionality:

The screen has 3 boxes with nothing in them. When the user clicks on play, each box, one by one get populated by a random number between 0 to 9. If all the three boxes have the same number the user hits a jackpot/wins (gets a score) and the difficulty of the game increases, if not he can try again. With each difficulty level the score attained for winning increases in a parabolic fashion.

Notes:

  • Slot Animation: Each of the boxes should emulate the spinning of a slot machine by quickly rotating the numbers before one steady value is populated.
  • Difficulty Level of the game: It is a number. Higher the number lower the probability of hitting the jackpot.
  • Winning Condition: All the 3 boxes show the same number.
